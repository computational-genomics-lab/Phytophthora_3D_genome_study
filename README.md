# Phytophthora_3D_genome_study
Chromosome scale assembly was generated using illumina, ONT and Hi-C technology. Three dimentional genome study was performed using Hi-C data. all codes and raw files used in this study is available in this repo. Updates may follow.

All Raw data for each plot is available under the figurewise folders. Codes used for the plotting is also available under Figure_wise_code_used_for_plotting folder.
